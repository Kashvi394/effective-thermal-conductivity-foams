# Influence of Boundary Conditions on Effective Thermal Conductivity of Foams

This repository contains preliminary work for a course project in *Micromechanics of Materials*, focused on understanding how boundary conditions influence the numerical computation of effective thermal conductivity in porous materials.

## 📌 Overview

The effective thermal conductivity of heterogeneous materials such as foams is highly sensitive to the choice of boundary conditions used in computational homogenization.

This project investigates the impact of:
- Dirichlet boundary conditions (UTG)
- Neumann boundary conditions (UHF)
- Periodic boundary conditions
- Mixed boundary conditions

on the estimation of effective thermal conductivity.

## Methodology

- Generation of foam microstructures using **Python / PuMA**
- Finite Element Analysis (FEA) using **Abaqus**
- Application of different thermal boundary conditions
- Extraction and comparison of apparent vs. effective properties
